To use the make file simply type in make and the program will compile
The names of the files include:
cheaters.cpp (the driver of the lab)
chunker.cpp
chunker.h
files.cpp
files.h
numArray.cpp
numArray.h
hashTable.cpp
hashTable.h

To run the program please type in ./plagiarismCatcher then your path 
to the file followed by the least number of similarities
(example)

./plagiarismCatcher path/to/files 6 200
